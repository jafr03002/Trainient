Progression - confirmed

Same rule as PR tracking: estimated 1RM via the Epley formula, per exercise, per set. For each exercise logged in both this week and last week, compare best e1RM week-over-week. Rank the top 3 improved exercises on the card (shown above). Roll the rest up into two headline percentages - % of tracked lifts that progressed and % that regressed - vs. last week. Total volume (sets) stays visible but demoted to a small secondary figure, never the headline number.

Backend note: this needs a new aggregation - /progress/prs already computes e1RM per exercise, but nothing currently diffs it week-over-week. That's new work in progress.ts, not a reuse of an existing route.

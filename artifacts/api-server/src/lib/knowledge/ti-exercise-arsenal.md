### Trainient Exercise Arsenal

INTRO:

- This is a exercise list per muscle groups that is dedicated to the program generation of a clients AI program.
- The exercise arsenal is supposed to be used as a standard 'base' with common movements that fit into most of the clients programs
- It is not used as a strict list of exercieses that the AI has to use to generate a program, rather a reference
- It is prefered to use the exercise names that i'll list. This is so that the name can be standardized across client profiles.
- Each exercise on this list will have a dedicated "how to preform this exerice" info sub page that will explain to the user what the exercise aims to target
along with how to set up and other general tips.
- This list will also include blabla
- The exercise arsenal are mostly 'Full Gym' only for now.
